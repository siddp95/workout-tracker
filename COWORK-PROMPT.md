# Cowork Kickoff Prompt

Copy and paste this into Claude at the start of any new Cowork session:

---

Read MEMORY.md and then let me know what we're working on and the current state of the project. I may have feedback or changes to make, or I may want to continue building new features.

---

That's it. Claude will read the memory file, summarize the current state, and be ready to continue.
